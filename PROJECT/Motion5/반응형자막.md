#### 반응형자막

```
참고 URI링크
https://www.youtube.com/watch?v=wbuystsB2kQ
용도 : 어떤 영상이던 
특정한 백그라운드가 깔린 반응형자막이 필요하다 이때 필요한건 예능형 자막
원하는 백그라운드에 자막을 깔고 영상상황에 맞는 자막을 만들어서 넣는다. 끝.
1. 5개항목중 Text를 선택한다.
2. 왼쪽 메뉴에 화면을 Fit 으로 맞추고 view 에서 safe Zones 선택한다.
3. 아래쪽에 도형중 rectangle 선택하여 safe zone 박스영역으로 범위를선택하여그린다.
4. 좌측메뉴중 "Inspector" 메뉴를 선택하여 Fill Color 선택후 색상을 변경한다.
5. 오른쪽 메뉴에 선택영역이 변경된 색상 255,255,255 검정으로 변경되었다.
6. 기존 text 삭제한다.


```
